    Worksheet Section 2.1 - Vectors and Linear Combinations Part 2
    

    Objectives

    Make sure you can answer the following questions:
    🔗
    • What is the connection between linear combinations and solutions to linear systems?
      🔗
      🔗
    🔗

    1.

    Let’s look at the vectors
    \begin{equation*} \vvec_1 = \fourvec{4}021, \vvec_2 = \fourvec1{-3}31,\vvec_3 = \fourvec{-2}110, \bvec = \fourvec012{-2}. \end{equation*}
    🔗

    (a)

    One thing we can do is ask if \(\bvec\) can be expressed as a linear combination of \(\vvec_1, \vvec_2, \vvec_3\) with weights \(c_1, c_2, c_3\text{.}\) Write out the equation for that linear combination.
    🔗
    🔗

    (b)

    We can rephrase this question by writing a linear system for the weights \(c_1, c_2, c_3\text{.}\) Solve this linear system (use Sage!).
    🔗
    🔗

    (c)

    Can \(\bvec\) be written as a linear combination of \(\vvec_1, \vvec_2, \vvec_3\text{?}\)
    🔗
    🔗
    🔗

    2.

    Consider the following linear system:
    \begin{equation*} \begin{array}{ccccc} 3x_1 \amp+ 2x_1 \amp- x_3 \amp=\amp 4\\ x_1\amp\amp + 2x_3 \amp=\amp 0\\ -x_1\amp-x_2\amp3x_3\amp=\amp1 \end{array} \end{equation*}
    🔗

    (a)

    Identify the vectors \(\vvec_1, \vvec_2, \vvec_3,\) and \(\bvec\text{.}\)
    🔗
    🔗

    (b)

    Rephrase the question ``Is this linear system consistent?" by asking ``Can \(\bvec\) be expressed as a linear combination of \(\vvec_1, \vvec_2,\) and \(\vvec_3\text{?}\)" What’s the answer to this question?
    🔗
    🔗
    🔗

    3.

    In previous math classes, you have probably seen the fact that, if we are given two points in the plane, then there is a unique line passing through both of them. We will begin with four points as shown on the left below, and try to find a polynomial that passes through these four points as shown on the right below.
    🔗
    \includegraphics[scale=.4]{polypic}
    🔗

    (a)

    A degree three polynomial can be written
    \begin{equation*} p(x) = a+bx+cx^2+dx^3 \end{equation*}
    where \(a, b, c,\) and \(d\) are coefficients that we would like to determine. Since we want the polynomial to pass through the point \((3,1)\text{,}\) we should require that
    \begin{equation*} p(3) = a+3b + 9c + 27d = 1 \end{equation*}
    and thus we obtain a linear equation for coefficients \(a, b, c\text{,}\) and \(d\text{.}\) Write the three other linear equations by requiring that the polynomial passes through the four points in the graph on the previous page.
    🔗
    🔗

    (b)

    Identify the vectors \(\vvec_1, \vvec_2, \vvec_3, \vvec_4\) and \(\bvec\text{.}\)
    🔗
    🔗

    (c)

    Rephrase the question ``Is this linear system consistent?" by asking ``Can \(\bvec\) be expressed as a linear combination of \(\vvec_1, \vvec_2, \vvec_3\) and \(\vvec_4\text{?}\)" What’s the answer to this question?
    🔗
    🔗

    (d)

    What if we added the point \((4,3)\text{?}\) How would the vectors change? Would the answer to the last question change?
    🔗
    🔗
